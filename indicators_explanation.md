# 方差分析结果指标详解

本文档对 Excel 中每个 Sheet 的所有指标逐一解释其**用途**与**结果含义**，并结合本研究（分型 / 分期）的实际数据进行说明。

---

## Sheet 1 — Descriptive Statistics（描述统计）

描述统计是所有统计分析的基础，用于概括每组数据的集中趋势和离散程度。

| 指标 | 用途 | 结果含义 |
|------|------|----------|
| **N** | 样本量 | 该组的有效观测值个数。样本量越大，统计检验的功效越高。 |
| **Mean** | 算术平均值 | 该组数据的集中趋势代表值。各组 Mean 的差异是方差分析比较的核心目标。 |
| **SD** (Standard Deviation) | 标准差 | 衡量数据围绕均值的离散程度。SD 越大，组内数据变异越大。 |
| **Median** | 中位数 | 将数据按大小排列后位于正中间的值。当数据存在偏态或异常值时，中位数比均值更稳健。 |
| **Min / Max** | 最小值 / 最大值 | 数据的极端范围。可用于判断是否存在异常值（outlier）。 |
| **Q1 / Q3** | 第一四分位数 / 第三四分位数 | 分别表示数据从小到大排列后第 25% 和第 75% 处的值。Q3 - Q1 = IQR（四分位距），是衡量离散度的稳健指标。 |

### 本研究结果解读

- **分型数据**：Normal 组均值最低（1.87），Her2 组最高（6.52），5 组均值差异明显。
- **分期数据**：Control 组均值最低（1.87），Stage I~IV 均值均在 5.7~5.9 之间，彼此非常接近。

## Sheet 2 — Normality Test（正态性检验）

正态性检验用于判断每组数据是否符合正态分布，这是选择参数检验（ANOVA）还是非参数检验（Kruskal-Wallis）的前提条件。

| 指标 | 用途 | 结果含义 |
|------|------|----------|
| **W_statistic** | Shapiro-Wilk W 统计量 | 取值范围 0~1。越接近 1 表示数据越接近正态分布；越偏离 1 表示偏离正态越严重。 |
| **p_value** | 显著性 p 值 | 原假设 H₀：数据服从正态分布。**p > 0.05** → 不拒绝 H₀，认为数据近似正态；**p ≤ 0.05** → 拒绝 H₀，数据显著偏离正态分布。 |
| **Normal_alpha_0.05** | 判定结果 | Yes = 通过正态性检验，No = 未通过。 |

### 为什么要做正态性检验？

单因素方差分析（ANOVA）的前提假设之一是各组数据服从正态分布。如果不满足，应改用 Kruskal-Wallis 等非参数方法。

### 本研究结果解读

- **分型数据**：5 组均未通过正态性检验（p 均远小于 0.05）→ 应采用非参数方法。
- **分期数据**：仅 Stage IV 通过（p = 0.14），其余 4 组均不满足正态性 → 应采用非参数方法。

---

## Sheet 3 — Levene's Test（方差齐性检验）

方差齐性（homogeneity of variance）是 ANOVA 的另一个关键前提假设：各组的方差应大致相等。

| 指标 | 用途 | 结果含义 |
|------|------|----------|
| **Statistic** | Levene 检验统计量 | 基于各组数据偏差绝对值的 F 检验。统计量越大，说明各组方差差异越大。 |
| **p_value** | 显著性 p 值 | H₀：各组方差相等。**p > 0.05** → 方差齐性成立；**p ≤ 0.05** → 方差不齐，ANOVA 结论可靠性下降，应使用 Welch ANOVA 或非参数方法。 |
| **Equal_variance_alpha_0.05** | 判定结果 | Yes = 方差齐性成立，No = 方差不齐。 |

### 为什么选 Levene 而非 Bartlett？

Bartlett 检验对非正态数据非常敏感，而 Levene 检验对偏态分布更为稳健，适用范围更广。

### 本研究结果解读

- **分型数据**：Levene p = 5.57e-21 → 方差**极度不齐**。
- **分期数据**：Levene p = 0.017 → 方差**不齐**（但程度较轻）。

---

## Sheet 4 — One-Way ANOVA（单因素方差分析）

ANOVA 是检验"三组及以上的均值是否全部相等"的核心方法。其原理是将数据的总变异分解为组间变异和组内变异。

| 指标 | 用途 | 结果含义 |
|------|------|----------|
| **Source** | 变异来源 | Between Groups = 组间变异（因分组产生的差异）；Within Groups = 组内变异（随机误差）；Total = 总变异。 |
| **SS** (Sum of Squares) | 离差平方和 | 衡量变异的大小。SS_between 越大 / SS_within 越小，说明组间差异越显著。 |
| **df** (degrees of freedom) | 自由度 | Between: k-1（k = 组数）；Within: N-k（N = 总样本量）。 |
| **MS** (Mean Square) | 均方 | MS = SS / df，即单位自由度的变异量。F 值 = MS_between / MS_within。 |
| **F** | F 统计量 | 组间均方与组内均方的比值。**F 越大**，说明组间差异相对于组内随机波动越大，越可能存在真实差异。 |
| **p_value** | 显著性 p 值 | H₀：所有组的总体均值相等。**p < 0.05** → 至少有一对组间存在显著差异（但不能说明是哪一对，需要事后检验）。 |
| **Significance** | 显著性标记 | \*\*\* (p<0.001), \*\* (p<0.01), \* (p<0.05), ns (不显著) |
| **Eta_squared** (η²) | 效应量 | η² = SS_between / SS_total，表示组间差异占总变异的比例。**0.01 = 小效应，0.06 = 中等效应，0.14+ = 大效应**。 |

### 本研究结果解读

- **分型数据**：F = 2087.50, p < 0.001, η² = 0.6336 → 组间差异解释了 63.4% 的总变异，**效应极大**。
- **分期数据**：F = 1648.26, p < 0.001, η² = 0.5750 → 组间差异解释了 57.5% 的总变异，**效应极大**。

> 注意：因为正态性和方差齐性均不满足，ANOVA 的 F 检验结果仅作参考，正式结论以 Kruskal-Wallis 为准。

---

## Sheet 5 — Kruskal-Wallis Test（克鲁斯卡尔-沃利斯检验）

Kruskal-Wallis 是单因素 ANOVA 的**非参数替代方法**。它不要求数据服从正态分布，也不要求方差齐性，而是基于秩次（rank）进行比较。

| 指标 | 用途 | 结果含义 |
|------|------|----------|
| **H_statistic** | H 统计量（卡方近似） | 类似于 F 值的非参数版本。H 越大，各组秩次分布的差异越大。 |
| **p_value** | 显著性 p 值 | H₀：各组的总体分布相同（中位数相等）。**p < 0.05** → 至少有一对组间分布不同。 |
| **Significance** | 显著性标记 | 同上。 |

### 什么时候用 Kruskal-Wallis？

当数据不满足正态性 **或** 方差不齐时，应以 Kruskal-Wallis 作为主要分析方法。本研究两个数据集均属此情况。

### 本研究结果解读

- **分型数据**：H = 2920.66, p < 0.001 → 5 组分布**极度显著不同**。
- **分期数据**：H = 2544.71, p < 0.001 → 5 组分布**极度显著不同**。

---

## Sheet 6a — Pairwise t-test（两两 t 检验）

ANOVA / Kruskal-Wallis 仅能告诉我们"组间存在差异"，但不能告诉我们"哪两组之间有差异"。事后两两比较（post-hoc pairwise comparison）解决这一问题。

本 Sheet 使用 **Welch's t-test**（不假设等方差的 t 检验）进行所有 C(5,2)=10 对比较。

| 指标 | 用途 | 结果含义 |
|------|------|----------|
| **Group_1 / Group_2** | 比较的两组 | 10 个成对组合。 |
| **t_statistic** | t 统计量 | 两组均值差异相对于标准误的比值。\|t\| 越大，差异越显著。正值表示 Group_1 均值高于 Group_2，负值反之。 |
| **p_raw** | 原始 p 值 | 单次检验的 p 值，未经多重比较校正。 |
| **p_adjusted_Bonferroni** | Bonferroni 校正后 p 值 | p_adj = p_raw × 比较次数（10）。Bonferroni 校正是最保守的多重比较校正方法，用于控制家族错误率（FWER）。校正后 **p_adj < 0.05** 方可认为显著。 |
| **Significance** | 显著性标记 | 基于校正后 p 值判定。 |

### 为什么需要 Bonferroni 校正？

进行 10 次比较时，即使各组真实均值相同，也有约 40% 的概率至少出现一次"假阳性"。Bonferroni 校正通过将 α 水平除以比较次数来控制这一风险。

### 本研究结果解读

- **分型数据**：除 Her2 vs LumA (p_adj=0.19, ns) 外，其余 9 对均显著 (p < 0.001)。
- **分期数据**：Control vs Stage I~IV 均极显著 (p < 0.001)；Stage I~IV 之间仅 Stage II vs Stage III 边缘显著 (p_adj=0.008, \*\*)，其余均不显著。

---

## Sheet 6b — Pairwise Mann-Whitney U（两两 Mann-Whitney U 检验）

Mann-Whitney U 检验是 t 检验的**非参数替代**，基于秩次比较两组数据的分布是否相同。因为本研究数据不满足正态性，此 Sheet 的结果是**最终推荐的事后比较依据**。

| 指标 | 用途 | 结果含义 |
|------|------|----------|
| **Group_1 / Group_2** | 比较的两组 | 同上。 |
| **U_statistic** | U 统计量 | 衡量两组秩次重叠的程度。U 值的大小本身需结合样本量解读；通常关注 p 值即可。 |
| **p_raw** | 原始 p 值 | 双侧检验的 p 值。 |
| **p_adjusted_Bonferroni** | Bonferroni 校正后 p 值 | 同上，p_raw × 10。 |
| **Significance** | 显著性标记 | 基于校正后 p 值。 |

### 本研究结果解读（最终推荐结论）

**分型数据**：

| 比较 | p_adj | 显著性 | 解读 |
|------|------:|:------:|------|
| Normal vs Basal | <0.001 | \*\*\* | Normal 指标值显著低于 Basal |
| Normal vs Her2 | <0.001 | \*\*\* | Normal 指标值显著低于 Her2 |
| Normal vs LumA | <0.001 | \*\*\* | Normal 指标值显著低于 LumA |
| Normal vs LumB | <0.001 | \*\*\* | Normal 指标值显著低于 LumB |
| Basal vs Her2 | <0.001 | \*\*\* | Basal 指标值显著低于 Her2 |
| Basal vs LumA | <0.001 | \*\*\* | Basal 指标值显著低于 LumA |
| Basal vs LumB | <0.001 | \*\*\* | Basal 指标值显著低于 LumB |
| Her2 vs LumA | 1.000 | ns | Her2 与 LumA 无显著差异 |
| Her2 vs LumB | 0.004 | \*\* | Her2 指标值略高于 LumB |
| LumA vs LumB | <0.001 | \*\*\* | LumA 指标值略高于 LumB |

**结论**：指标值由低到高排列为 Normal < Basal < LumB < LumA ≈ Her2，各分型间差异显著。

**分期数据**：

| 比较 | p_adj | 显著性 | 解读 |
|------|------:|:------:|------|
| Control vs Stage I~IV | <0.001 | \*\*\* | 对照组指标值显著低于所有肿瘤分期 |
| Stage I~IV 之间 | >0.05 | ns | 不同分期之间无显著差异 |

**结论**：主要差异存在于正常对照与肿瘤之间，肿瘤分期（I~IV）内部指标值无显著差异。

## Sheet 7 — Summary（汇总）

| 指标 | 用途 | 结果含义 |
|------|------|----------|
| **Dataset** | 数据集名称 | 标识分析对象。 |
| **Number of groups** | 组数 | 本研究均为 5 组。 |
| **Total observations** | 总观测数 | 所有组的样本量之和。 |
| **All groups normal?** | 正态性判定 | 基于 Sheet 2 的综合结论。Yes = 全部通过，No = 至少一组未通过。 |
| **Equal variance?** | 方差齐性判定 | 基于 Sheet 3 的结论。 |
| **Recommended method** | 推荐统计方法 | 根据正态性和方差齐性的检验结果，自动推荐最合适的方法。 |
| **Key result** | 核心统计结果 | 推荐方法对应的检验统计量和 p 值。 |
| **Effect size (Eta-squared)** | 效应量 | 见 Sheet 4 解释，η² 越大表示组间差异占总变异比例越高。 |

### 方法选择逻辑

```
正态性 + 方差齐 → One-Way ANOVA + Bonferroni/Tukey 事后检验
正态性 + 方差不齐 → Welch ANOVA + Games-Howell 事后检验
非正态 → Kruskal-Wallis + Mann-Whitney U 事后检验  ← 本研究两个数据集均使用此方案
```

## 附：显著性标记说明

| 标记 | p 值范围 | 含义 |
|:----:|:--------:|------|
| \*\*\* | p < 0.001 | 极显著差异 |
| \*\* | 0.001 ≤ p < 0.01 | 高度显著差异 |
| \* | 0.01 ≤ p < 0.05 | 显著差异 |
| ns | p ≥ 0.05 | 无显著差异 (not significant) |

## 附：效应量（η²）参考标准

| η² 范围 | 效应大小 | 解读 |
|:--------:|:--------:|------|
| < 0.01 | 极小 | 组间差异可忽略 |
| 0.01 ~ 0.06 | 小 | 组间差异较小 |
| 0.06 ~ 0.14 | 中等 | 组间存在实际意义的差异 |
| > 0.14 | 大 | 组间差异非常显著 |

本研究分型 η² = 0.6336、分期 η² = 0.5750，均属于**极大效应**，说明组间差异是数据变异的主要来源。
